코틀린의 변수 선언 방식인 var과 val에 대한 정리!

기존 자바에서의 변수 선언 방식
```{.java}
int age = 28;
```

코틀린에서의 변수 선언 방식

{% highlight%}
val ageVal = 28
var ageVar = 28
{% endhighlight %}
